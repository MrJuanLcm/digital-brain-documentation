# 🔌 Plugins recomendados para Obsidian

---

## ⚡ Plugins esenciales

| Plugin | Por qué lo necesitas |
|---|---|
| 📅 **Periodic Notes** | Notas diarias/semanales automáticas |
| 🔗 **Dataview** | Consultas SQL-like sobre tus notas |
| 🔍 **Omnisearch** | Búsqueda rápida y profunda |
| 🏷️ **Tag Wrangler** | Gestionar etiquetas masivamente |

---

## 📦 Plugins útiles

| Plugin | Para qué sirve |
|---|---|
| 📎 **Paste URL into Selection** | Pega URLs como links formateados |
| 🖼️ **Excalidraw** | Diagramas y dibujos dentro de notas |
| ✅ **Kanban** | Tableros Kanban en markdown |
| 📊 **Charts** | Gráficos desde datos en notas |
| 🔄 **Obsidian Git** | Auto-commit y push del vault a Git |
| 📝 **Templater** | Plantillas avanzadas con scripts |
| 🧠 **Smart Connections** | Sugiere conexiones entre notas con IA |
| 🎯 **Tasks** | Gestión avanzada de tareas |
| 📐 **Mermaid** | Diagramas Mermaid nativos |

---

## ⚙️ Plugins para el Digital Brain

| Plugin | Por qué es clave |
|---|---|
| 🔌 **Obsidian Local REST API** | Expone el vault como API REST (alternativa al MCP) |
| 🤖 **MCP Plugin (community)** | Permite que herramientas externas se conecten |

---

## 🔧 Cómo instalar plugins

1. Abre **Obsidian**
2. Ve a **Configuración** → **Community plugins**
3. Desactiva **Safe mode** (Modo restringido)
4. Haz clic en **Browse** (Explorar)
5. Busca el plugin por nombre
6. Haz clic en **Install** (Instalar)
7. Luego **Enable** (Activar)
