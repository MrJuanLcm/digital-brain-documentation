# 🤝 Contribuir

¡Gracias por tu interés en mejorar el Digital Brain!

## Cómo contribuir

1. **Fork** el repositorio
2. Crea una rama: `git checkout -b mi-mejora`
3. Haz tus cambios
4. Asegúrate de que los archivos `.md` tengan buen formato
5. Commit con mensaje claro: `git commit -m "feat: descripción clara"`
6. Push: `git push origin mi-mejora`
7. Abre un **Pull Request**

## Guías

- 📝 Usa Markdown estándar
- 🎯 Un cambio por PR
- 🧪 Si agregas scripts, pruébalos antes
- 📖 Mantén la documentación clara y simple
