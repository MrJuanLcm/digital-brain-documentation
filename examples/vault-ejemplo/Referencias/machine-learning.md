# Machine Learning

> Rama de la inteligencia artificial que permite a sistemas aprender patrones a partir de datos sin ser programados explícitamente.

## Tipos principales

- **Supervisado:** Entrenamiento con datos etiquetados
- **No supervisado:** Encuentra patrones en datos sin etiquetar
- **Refuerzo:** Aprende mediante prueba y error con recompensas
